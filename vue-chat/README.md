# Vue Realtime Chat

Aplicación de chat en tiempo real desarrollada con **Vue 3 + TypeScript + Socket.io**.

---

# Funcionalidades

* Conexión en tiempo real mediante **Socket.io**
* Envío y recepción de mensajes instantáneos
* Indicador de estado de conexión
* Visualización del historial de mensajes
* Scroll automático cuando llegan nuevos mensajes
* Persistencia de mensajes en **localStorage**
* Persistencia del usuario
* Avatar simple basado en la inicial del usuario
* Limpieza del historial de mensajes

---

# Tecnologías utilizadas

* **Vue 3**
* **TypeScript**
* **Composition API**
* **Vite**
* **Socket.io-client**
* **Pinia**

---

# Ejecución del proyecto

## 1. Instalar dependencias

```
npm install
npm run dev
```

## 2. Configurar variables de entorno

Copiar:

.env.example → .env

La aplicación estará disponible en:
```
http://localhost:5174
```
